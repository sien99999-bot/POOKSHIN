## 1. 제품 개요

### 앱명
**POOKSHIN** (푹신)

### 한 줄 설명
AI 마스코트 '와앙이'가 사용자의 빵 취향을 분석해 주변 베이커리와 메뉴를 추천하는 푸드 큐레이션 앱

### 핵심 가치
| 가치 | 설명 |
|------|------|
| 취향 기반 추천 | 식감·재료·빵 종류 데이터를 수집해 개인화 추천 제공 |
| POKI 액션 | 추천 메뉴를 '포키(저장/좋아요)'하면 길찾기까지 원스톱 연결 |
| 식감 데이터화 | 빵의 식감을 바삭/촉촉/버터/쫄깃/담백/꾸덕 6축 정량 지표로 시각화 |
| 마스코트 UX | 와앙이 캐릭터가 전 플로우에서 친근하게 안내 |
| 취향 누적 분석 | 포키·리뷰 누적으로 DNA 요약·레이더 차트·트렌드 그래프 제공 |

### 지원 플랫폼
- iOS 우선 출시
- 기준 디바이스: iPhone 14 Pro / 15 (430pt, Safe Area Top 55px / Bottom 34px)

## 2. 사용자 & 목표

### 타겟 사용자
- 20~30대 베이커리·카페 방문을 즐기는 사용자
- 새로운 빵집을 발견하고 싶지만 선택이 어려운 사람
- 자신의 식감 취향을 데이터로 확인하고 싶은 사람

### 사용자 목표
1. "오늘 어떤 빵 먹지?" 결정 피로를 없애고 싶다
2. 내 취향에 딱 맞는 주변 베이커리를 빠르게 찾고 싶다
3. 내가 먹은 빵들의 식감 취향 패턴을 알고 싶다

### 비즈니스 목표
1. 사용자 취향 데이터 수집 및 고도화
2. 베이커리 파트너십 연계 (길찾기 → 방문 전환율 측정)
3. 리뷰 기반 커뮤니티 형성

```
  └→ [스플래시] 로딩 (~2초)
       └→ [온보딩 01] 감성 인트로 (자동 전환)
            └→ [온보딩 02] 와앙이 소개
                 └→ [시작하기] ────────────────────────────────┐
                      └→ [온보딩 03] 빵 종류 선택               │
                           └→ [다음]                           │
                                └→ [온보딩 04] 식감 선택        │ 이전 버튼으로
                                     └→ [다음]                 │ 각 단계 복귀
                                          └→ [온보딩 05] 세부 취향 선택
                                               └→ [다음] → 취향 데이터 서버 저장
                                                    └→ [로그인]
                                                         └→ [소셜 OAuth 성공]
                                                              └→ [홈 화면]
```

### 7.2 핵심 추천 플로우
```
[홈 화면] 추천 카드 표시
  ├→ [NEXT] ─────────────────────────────→ 다음 추천 메뉴 로드 (슬라이드 전환)
  │
  ├→ [POKI] ─────────────────────────────→ [포키 완료 페이지]
  │                                              └→ [길 찾기] → [지도 (가게 포커스)]
  │
  └→ [상세보기] ──────────────────────────→ [상세 페이지] (3268px 스크롤)
       ├→ 히어로 스와이프 (추가 사진)
       ├→ [같은 가게 다른 메뉴] → 가게 메뉴 리스트
       └→ [리뷰보기] → [리뷰보기 화면]
                           └→ [리뷰 더보기] → 페이지네이션
```

### 7.3 리뷰 작성 플로우
```
[마이페이지] 또는 [리뷰보기 화면]
  └→ [리뷰하기 >] 진입
       └→ [리뷰하기] 화면
            ├→ 포키 빵 탭 선택
            ├→ 별점 탭 선택
            ├→ 식감 세그먼트 바 설정
            ├→ 한줄 리뷰 텍스트 입력 (선택)
            ├→ 사진 첨부 (선택)
            └→ [리뷰 등록하기]
                 ├→ API 성공 → [리뷰 등록 완료 팝업] → [확인] → 마이페이지 복귀
                 └→ API 실패 → 에러 토스트 메시지 표시
```

### 7.4 취향 데이터 확인 플로우
```
[마이페이지]
  └→ [내 취향 데이터 보기 >]
       └→ [내 취향 데이터] 화면 (풀 스크롤)
            ├→ DNA 요약 (태그 클라우드 + 한줄 요약)
            ├→ 취향 밸런스 (레이더 차트)
            ├→ 내 취향 트렌드 (도넛 그래프 + 꺾은선 그래프)
            ├→ 와앙이의 코멘트 (개인화 메시지)
            └→ 추천 빵 가로 스크롤
```

### 7.5 지도 탐색 플로우
```
[Tab Bar MAP 탭] or [포키 완료 → 길 찾기]
  └→ [지도] 화면
       ├→ 지도 마커 탭 → 리스트 해당 가게 스크롤 포커스
       ├→ [길찾기] → 외부 지도 앱 (카카오맵 / 네이버맵) 딥링크
       └→ 북마크 아이콘 탭 → 찜 토글
```

## 8. 비기능 요구사항

### 성능
| 항목 | 기준 |
|------|------|
| 앱 콜드 스타트 | 3초 이내 홈 화면 진입 |
| 추천 응답 | [NEXT] 탭 후 1초 이내 다음 카드 표시 |
| 이미지 로딩 | skeleton placeholder (`#D9D9D9`) 표시 후 비동기 로드 |
| 지도 로딩 | 현재 위치 중심 3초 이내 로드 |
| 상세 페이지 스크롤 | 3268px 길이 — 섹션별 LazyLoad 적용 필수 |

### 보안
- 소셜 로그인 OAuth 2.0 표준 준수
- 개인정보 (이메일, 생년월일) 암호화 저장
- 리뷰 닉네임 마스킹: "와앙*****" 형식

### 접근성
- 모든 이미지에 alt 텍스트 제공
- 최소 터치 영역 44×44pt (iOS HIG 기준)
- 다크모드: 설정에서 토글 스위치로 전환 (현재 활성 상태 디자인 존재)

### 오프라인
- 네트워크 없을 시 명확한 에러 상태 UI 표시
- 마지막 추천 결과 캐싱 (1시간)

### iOS Safe Area
- Top: 55px / Bottom: 34px (Home Indicator 포함)
- Tab Bar 하단 padding 반드시 Safe Area 반영
